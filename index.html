MiniFrameWork

Mijn Framework Bestaat uit 3 bestanden:
  • MiniFramework-Colors
  • MiniFramework-Grid
  • MiniframeWork

MiniframeWork-Colors heeft de volgende klasses:

  • text-Kleur bv. text-Black altijd de kleur met een hoofdletter typen.
  • bg-kleur bv. bg-Black altijd de kleur met een hoofdletter typen.

Voor verdere info welke kleuren in de framework zitten kan je naar deze link gaan kleuren

MiniframeWork-Grid werkt hetzelfde als bootstrap grid het verschil met mijn grid syteem is namelijk dat je ook de hoogte van de rijen kan manipuleren. Dit doe je op de zelfde manier als je de kolommen manipuleert bv. row-6 row-6 voor twee dezelfde kolommen

  • text-fat bold text met capitalize
  • text-center center de tekst
  • btn standaard knop
